# Описание
Данное приложение является образовательным генератором-решателем лабиринтов со следующими функциями:
1.	Генерация лабиринта с настройкой параметров: пользователь может сам задавать параметры генерируемого лабиринта (размер, плотность стен, различные алгоритмы генерации).
2.	Выбор и запуск алгоритма решения: выбор необходимого алгоритма из списка доступных, его запуск и визуализация решения.
3.	Сравнение алгоритмов решений: запуск нескольких алгоритмов сразу и их сравнение. Каждый отдельный алгоритм помечается своим цветом (мо-жет происходить наложение цветов, если присутствует один и тот же путь). Статистика (за какое время был решен лабиринт, за какое количество клеток и прочее) выводится в отдельной таблице прямо в приложении.
4.	Прохождение созданного лабиринта пользователем: пользователь может сам пройти сгенерированный лабиринт. После прохождения, можно сравнить путь и статистику, пройденный пользователем, с теми, которые будут у алгоритмов.
5.	Просмотр описания алгоритмов: должен присутствовать отдельный раздел с описанием всех используемых в программе алгоритмов.

# Методы использования
- <b><u>Windows</b></u>

Загрузите [архив из релизов](https://github.com/GinDoyr/MazeGenSolver/releases/download/PreRelease/MazeGenSolver_win.rar), распакуйте и запустите файл MazeGenSolver.exe

- <b><u>Linux</b></u>

1. Загрузите [файл .AppImage из релизов](ссылку!!!) 
2. Сделайте файл исполняемым, выполнив в терминале команду:
```
chmod +x MazeGenSolver-*.AppImage
```
3. Приложение готово к запуску. Можно запускать, например, с помощью команды
```
./MazeGenSolver-*.AppImage
```
