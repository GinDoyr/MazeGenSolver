# Описание
Данное приложение является образовательным генератором-решателем лабиринтов со следующими функциями:
1.	**Генерация лабиринта с настройкой параметров**
   
    Пользователь может сам задавать параметры генерируемого лабиринта (размер, плотность стен, различные алгоритмы генерации).
2.	**Выбор и запуск алгоритма решения**
   
    Выбор необходимого алгоритма из списка доступных, его запуск и визуализация решения.
3.	**Сравнение алгоритмов решений**
   
    Запуск нескольких алгоритмов сразу и их сравнение. Каждый отдельный алгоритм помечается своим цветом (может происходить наложение цветов, если присутствует один и тот же путь). Статистика (за какое время был решен лабиринт, за какое количество клеток и прочее) выводится в отдельной таблице прямо в приложении.
4.	**Прохождение созданного лабиринта пользователем**
   
    Пользователь может сам пройти сгенерированный лабиринт. После прохождения, можно сравнить путь и статистику, пройденный пользователем, с теми, которые будут у алгоритмов.
5.	**Просмотр описания алгоритмов**
   
    Должен присутствовать отдельный раздел с описанием всех используемых в программе алгоритмов.

# Методы использования
- **Windows**

Загрузите [архив из релизов](https://github.com/GinDoyr/MazeGenSolver/releases/download/Release/MazeGenSolver_win.rar), распакуйте и запустите файл MazeGenSolver.exe

- **Linux**

1. Загрузите [файл .AppImage из релизов](https://github.com/GinDoyr/MazeGenSolver/releases/download/Release/MazeGenSolver-x86_64.AppImage) 
2. Сделайте файл исполняемым, выполнив в терминале команду:
```
chmod +x MazeGenSolver-*.AppImage
```
3. Приложение готово к запуску. Можно запускать, например, с помощью команды
```
./MazeGenSolver-*.AppImage
```
